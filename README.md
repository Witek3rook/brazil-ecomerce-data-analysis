# Brazilian E-Commerce (Olist) Performance Analysis 📊
End-to-end data analysis of Brazilian E-Commerce (Olist). Extracting insights using SQL (PostgreSQL) and visualizing business performance &amp; logistical bottlenecks in MS Excel.




## 📌 Business Problem
Olist (a Brazilian e-commerce platform) needed a high-level overview of their sales, customer retention, and logistical bottlenecks. 

## 🛠️ Tools Used
* **SQL (PostgreSQL / MySQL):** Data extraction, merging multiple tables (`JOIN`), and creating logical categories (`CASE WHEN`).
* **MS Excel:** Data visualization, interactive dashboard creation, and data modeling.

## 💡 Key Insights (Wnioski)
1. **São Paulo Dominates:** Over 40,000 orders originate from the SP state, making it the core market.
2. **Top Selling Categories:** Health & Beauty (`beleza_saude`) and Watches/Gifts are generating the most revenue.
3. **Logistics is Killing the Vibe (and Scores!):** We analyzed over 96,000 orders. Packages delivered on time maintain a solid **4.29** average review score, while late deliveries plummet to **2.57**, driving customers away.
